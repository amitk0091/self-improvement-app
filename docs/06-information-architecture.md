# 06 — Information Architecture

```
Home (Today)
├─ Daily Diary (quick log)
├─ Checklists (templates + today)
├─ Feelings (quick mood)
├─ Overthinking (park + history)
├─ Analytics (SWE, fitness, sleep, mood, streaks)
├─ AI Review (weekly insights)
└─ Settings (goals, reminders, export)
```
Navigation: bottom tab bar (mobile) / sidebar (desktop). Today is default. Streak shown in header. Secondary goals = tags inside diary, not nav items.
