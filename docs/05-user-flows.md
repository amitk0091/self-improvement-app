# 05 — User Flows

## Daily log (core, ≤60s)
1. Open app → Today screen. 2. Defaults pre-filled. 3. Toggle exercise/bath/meditate. 4. Slide mood/sleep, type SWE hrs. 5. One-line went-well/improve. 6. Autosave → streak +1.

## Overthinking park
1. Tap "Park a thought" → textbox + category. 2. Save → "Logged. Let it go." 3. Weekly review surfaces patterns.

## Checklist
1. Pick template → daily instance. 2. Tick items → % bar. 3. Auto-reset next day.

## Weekly AI review
1. Sunday: cron runs. 2. Summarize 7 days → Gemini. 3. Show strengths/weakness/pattern + 1 action. 4. User accepts → becomes next-week focus.

## Onboarding
1. Magic link login. 2. Set 2 goals + targets. 3. Set reminder time. 4. First log.
