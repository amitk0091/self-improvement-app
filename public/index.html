Personal OS

Self-Improvement App

A comprehensive personal development application built with Next.js 13+ and Supabase.

Daily Journaling

Track your SWE hours, sleep quality, mood, motivation and more.

AI Insights

Weekly review powered by AI analysis of your data patterns.

Analytics

Visualize your progress over time with interactive charts.

Getting Started

  1. Clone the repository and install dependencies
  2. Set up your Supabase account with the provided schema
  3. Configure environment variables
  4. Run the development server with npm run dev
